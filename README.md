# Teach

Teach es una plataforma educativa para gestionar cursos, aulas, Teams, contenidos, Fast Tests, entregas y evaluaciones.

## Estado del proyecto

Versión inicial en desarrollo.

## Tecnologías

- HTML
- Tailwind CSS
- JavaScript
- Firebase Authentication
- Cloud Firestore
- GitHub Pages

## Roles

- Superadmin
- Administrador institucional
- Profesor
- Estudiante
